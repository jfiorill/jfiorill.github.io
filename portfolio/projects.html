Crystal Clashers

This is a two player game with twelve different characters to choose from, created in Scratch. In a similar to dodgeball style, players with launch crystals at each other until one players health is completely gone. However, every character has a unique ability and are granted buffs as their health gets lower. Additionally, each character has their own special move that charges up over time.

Run Master

This is a mobile app created in Android Studio designed to track your runs. It has five different tabs, one as a pace calculator, one to suggest workouts for different length events, one to add and view activities you do, one with recent news, and another to switch between miles and kilometers.

NFL 2022-23 Season Simulator

This is a simulation of an NFL season made using Python in Command Prompt. It goes through every game then will take the top teams through to the playoffs. It updates the scores for each game by quarter to add some type of anticipation, as well as show the full standings whenever a week is completed. Additionally, there is code to make it so that the teams with a better record have a slightly better chance of winning
